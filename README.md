# Shoppers Stop app using Next JS

## Installation :wrench:

1. `cd next-js-app`
2. `npm install`
3. `npm run dev`

## Common Commands :computer:

1. `dev` Starts your machine development server [http://localhost:3000](http://localhost:3000)
2. `build` Runs build
3. `start` Runs next js server
